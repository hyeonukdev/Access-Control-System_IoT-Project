### 적용기술
OS : Ubuntu 16.04<br>
Framework: OpenCV 3.3<br>
Language : Python/HTML/JS/CSS<br>
Server : Apache/NAS<br>
Code : Git<br>

### 상세 내용
- 웹 서버
- 데이터베이스 관리

- 웹 서버 구축
  - 라즈베리파이 Apache 서버 구축
-  DB 구축
  - 라즈베리파이 MySQL 구축
  - iotDB 데이터베이스 생성
  - time, permit 속성 생성
- 출입이력 관리
  - 라즈베리파이 phpmyadmin 웹 제어
- 웹 설계 및 구현
  - 라즈베리파이 Apache 서버에서 동작
  - 실시간 스트리밍 출력
  - 출입이력 DB 출력
